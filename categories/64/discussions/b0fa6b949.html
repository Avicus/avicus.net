Mczone Skywars revival. by Shirokamii January 4, 2017 at 1:01 AM UTC

Hello there users!

Manny might not know me, or maybe you do!
Ive been with avicus since the birth, ive seen our keenan build a skywars server at the age of 14, creating one of the finest skywars server arround. but what made the Mczone skywars so great?

Well i can explain all of that.

1. easy.
 The map wasnt fancy, nor was it hard to get used to, it was a plain part of minecraft land lifted up into the air. combined with a few biomes.

2. Speed.
Due the map simplicity the gamestyle was, chaoticly fast. people had 30 seconds to choose a kit and go!

3. fun.
No one cared if you used an enderpearl and smashed into the void, Everyone did it! we used this gamemode to have some fun, try trickshots! if you failed, you just went into one of the other 15 server and try it again!

So maybe if we can recreate this feeling, our community can return to its old glory!

So, hereby i present :

Skywars 1.0 Mczones finest gamemode!
