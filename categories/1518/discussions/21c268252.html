[10/19/17] Custom Website Badges by ALM October 19, 2017 at 9:10 AM UTC

Hey all,

I would like to to first off start by apologizing for the lateness of this update and the odd time of this post. I've been up for a while writing an essay and this is the last thing I need to do before going to sleep.

ANYWAYS, here's the update!

As you might have noticed, a few of the icons around the site might look a bit different. A big thanks to  trevin for updating the website icon set to a more modern version. His work made this update possible.

The main part of the update, however, is the addition of custom forum badges for premium users. These appear next to your ranks on the forums and can be configured from your edit profile page. The higher tier ranks have access to more icons and colors. Gold only has the basic icon and color set, and Emerald/Diamond have more colors/icons respectively. There are even some sets not enabled for anyone right now, which will allow us to give these out as special prizes. Each user can only have one badge and only YOU can edit it.

Also, since this update was late, I have tacked on "good faith" feature. This was going to be released later but I wanted to make up for my lateness. The Nyan gadget is back in the gadget store under "Devices". Go grab it!

Also  Kashike is a dev again! He's going to be working on mostly backend stuff to make your experience more streamlined and efficient. Some of the stuff he does you won't see, but the changes he is making are making Avicus more structurally sound and easy to extend.

Thanks,
The Avicus Development Team
