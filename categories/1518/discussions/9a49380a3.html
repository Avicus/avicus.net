Hello everyone, Happy Forday!

Today we are announcing a long-requested feature, Premium Chat! This is a simple chat which cane used by all premium users in game to talk to each other. All donor ranks cause the chat, emeralds can use colors, and diamonds can use bold/underline/stikethrough. You can turn off the chat with a setting so that it will never become a hindrance.

To access that chat type /pr <message>
