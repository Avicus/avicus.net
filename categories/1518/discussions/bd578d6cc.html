[9/13/17] Private Messaging on Forums by ALM September 13, 2017 at 3:09 PM UTC

Hello all, it's Wednesday again!

Today we are releasing a long-requested feature, private messaging on the forums. This feature allows use to send private message to one (or multiple if they are emerald/diamond) from a page on the website. When the message is received, the user will be given an alert in the navigation bar and will receive the normal unread alerts message in game. Your inbox is located in the dropdown menu under your avatar.

A Note on Privacy:
Your messages, just like the in game ones you send, are private to only you and the sender. If a report is issued, evidence can be queried by a developer in special circumstances but is not readily available for common monitoring.

Abuse of this system can result in a web punishment.

Thanks,
ALM
