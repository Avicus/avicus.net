Nebula Player Sizes Adjustment by Jongh December 16, 2015 at 1:12 AM UTC

Avicus usually only has 1 or 2 servers running, and often only one of these is full. When a match cycles to a map with a smaller map size many players end up not being able to play, because the other server isn't full/online, and their friends are in the current server.

I was thinking maybe some team sizes could be adjusted or maps grouped so their player sizes are more uniform. Or, just remove the smaller maps and make Nebula have only maps with a large player count. Currently Nebula's map's player sizer fluctuate a bit, so if it were OCN it would sort of be like changing to a Capture-style server, where all the maps have 64 player size.

For me, if I can't join within a few minutes I'll usually just leave.

If there were more servers the answer to this question would just be 'find another server' but when Avicus really has only 1 full server running at a time this issue can be a huge turn off to non-donors.
