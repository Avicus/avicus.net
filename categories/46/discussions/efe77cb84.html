[SUG] 2015 Competitive Nebula Season! by PokerFace April 13, 2015 at 10:04 PM UTC

Hey everybody,
I've had quite a bit of experience with sports and standings and, seeing the NHL Stanley Cup post, I thought, why not bring that to Nebula? It could have a similar concept as the scoring in sports played today. However, it would require some coding and some planning to hold matches.

Scoring:
Win- 2 points
Loss- 0 points
Overtime/tie- 1 point

Basically, depending on the number of teams we have here, we could setup a really neat statistics page. Below are some things that should be included.

Team Statistics:
Win/Loss/Tie Ratio ex. (7-3-1)
Monuments Destroyed
Kills
Deaths
K/D Ratio
Standings compared to other teams.

Player Statistics:
Game played
*Note that players will not always be able to attend every game.
Monuments Destroyed
Kills
Deaths
K/D Ratio
Standings compared to all other players
*Note that you could have different standings filters such as, most monuments broken, highest K/D etc..

Game Scheduling:
Basically, a set schedule should be set perhaps 1 game every 2 weeks. It would also be essential to have the timezone information of all the teams to be able to coordinate a reasonable time for a match. If we could input some sort of html coding which would allow us to state the timezone of each member in a team, matches could be suited to a reasonable time. Perhaps, we could have seasons (each 3 months long).

Scorekeeping:
This would be an absolute necessity. There is this program called SportNGIN Live. It allows you to create teams and has a simple way of handling statistics. The whole plugin comes with everything you need such as the game schedule bar, statistics, etc. One of the downsides is that, the program was made for sports, not for video games. Perhaps they have a sports layout that might fit the game of Nebula however. If not, I believe many people here would be willing to contribute their coding skills (i hope). If this program somehow fits the standards of nebula, you would be able to create admin accounts that will allow people to scorekeep the game. Once you finalize the game, all statistics are automatically updated for you. I know this because, I scorekeep some Rec. League hockey games. If you would like an example of a layout, visit this website. http://www.kihawaii.com/
(It's a bit expensive so, custom coding may be the better option)

If money is not an issue, feel free to visit http://www.sportngin.com

Anyways, this is just an idea and I believe that if this were to work, this will bring in thousands of players. Competition is what brings people in. That's why sports are super popular and I believe the same concept could be achieved here. Hopefully this can be achieved and if you are reading this KyloCode, I thank you.

Any feedback/suggestions are welcome as long as you have reasoning for it.
Thanks!

-PokerFace

*1 more note. Perhaps, the top teams during the season can gain eligibility into the tournaments. Similar to the concept used in the NHL.
