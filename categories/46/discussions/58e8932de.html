Making Game Launchers by ParthDarthVader February 17, 2015 at 12:02 AM UTC

Hey guys! I'm a major coding geek so I decided to make a game launcher. I know, a little pathetic, considering that all the games we play have their own launcher. But I did it anyway. So to start off, let me show you an example of what I did. (This will be the template of the other launchers.)



I had actually started to make a TF2 launcher for convenience for me, but I decided not to anyway. Instead, as a test run, I made it run Steam.

Features
  • First off, when you hit the play button, it reads the text box to the right. That is where you paste in the file address. The "Browse" button just opens Windows explorer (Which reminds me, this is a program made for Windows. Sorry Mac users D:). You input the file location and when you press "Play", it will run the application.


  • There is a Web Browser. At the top, you will notice the TF2 website. You can navigate it while the launcher is open. Unfortunately, there is no back button, forward, refresh, etc. I can add them; it's just that I'm too lazy :P


Things that should be added

I would love to have suggestions and try to implement them. Please note that I will not be able to do everything you ask of.
  • Not having to go through explorer to copy and paste file location.
  • Suggestions!
