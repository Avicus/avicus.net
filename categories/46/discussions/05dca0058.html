Minecraft 1.10 Pre-Release 2 is now out! by Brokennz June 2, 2016 at 7:06 PM UTC

Yep, Minecraft 1.10 pre-release is now out and this means that in 3 or 1 months we will have a new Minecraft version.
This is the same case as the Minecraft 1.1 with 5 snapshots, 1.2 with 8 snapshots and finally 1.10 with 3 snapshots! It seems that Mojang didn't put a lot of effort in this lastest snapshots (like they did with the Bountiful Update)

Mojang released the Pre-Release 2, which have some bug fixes and changes from the last snapshot. (It doesn't seems that they will delay the release date of the Minecraft 1.10... let's wait to another week and see Official response!)

In this version, they added a few things like:
  • Polar Bears
  • Husk - New type of Zombie
  • Stray - New type of Skeleton
  • Magma Block
  • Auto-Jump
  • Nether wart Block
  • Bone Block
  • Red nether brick Block
  • Structure Block
  • New Villages
  • Changes in the AI
Noables changes in the Pre-Release 2:
  • Fixed more bugs
  • Added missing sounds
