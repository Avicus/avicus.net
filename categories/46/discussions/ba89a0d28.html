Pokemon Nuzlocke Challenge Thred! by Shoto_ June 19, 2014 at 5:06 PM UTC

Playing Pokemon over and over again gets kind of boring. :/ So what do I do? I go hardcore.
My rules:
1. Any Pokemon that faints MUST be released.
2. Only catch the first Pokemon encountered in each patch of grass.
3. Whiteout/blackout is GAME OVER
4. If last number of trainer ID is 1-3, start with grass type, 4-6 is fire type, 7-9 is water type, and 0 is player's choice. (I can't find ID in Red so I just pick Bulbasaur)
5. Rules don't start until Pokeballs are recived
6. Must nickname ALL Pokemon to form stronger bonds


Share your Nuzlocke stories here, and you can play Pokemon at www.playr.org
