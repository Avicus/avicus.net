Porkyyy August 13, 2014 at 11:08 PM UTC

1 - Download world edit mod for the version of minecraft that you are using.
2 - Go to your .minecraft.
3 - Find the version of minecraft that you want to use World Edit with.
4 - Double click, and copy the 2 files inside (.json and .jar).
5 - Create a new folder and name it something which you will remember (Caps matter!!).
6 - Paste the two files in there.
7 - Rename the files the same name that you named the folder.
8 - Watch the remainder of the video.
