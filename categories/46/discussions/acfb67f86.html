How to get Halo for PC by Rainstruck February 7, 2015 at 9:02 PM UTC

I'm sure that we all have a different game other than Minecraft that we would like to play. I have tons of them including, but not limited to, Smash Bros., Legend of Zelda, Mario Kart, and recently, Halo. Lets talk about Halo. I don't have an Xbox 360 or Xbox One, So that left me with one game Halo: Combat Evolved. This was the only game that was released on anything other than a console. So here we are with the improved FREE version of CE called... Halo Custion Edition!!

This game is just so fun!! I don't really have any other words to describe except of just pure joy and fun!! So you wanna know how to get it Halo on your fancy computation device you might be reading this thread on? Well guess what, I'm going to tell you. 

Here is the video that I followed to get it:
https://www.youtube.com/watch?v=_SyzTkaqaYI
However there are some new instructions in order to get it, and some other instructions to get it:
1.  You must have win.rar installed, no matter what computer you are on. You don't have to buy the full version you can just download the trial version
http://www.win-rar.com/start.html?&L=0
2. Download Halo with this link:
http://hce.halomaps.org/index.cfm?fid=410
3. Open Halo with win.rar and run the .exe file to install Halo.
4.  When you run Halo it will ask for a key, follow this video link and use ANY key code to get the game:
https://www.youtube.com/watch?v=xc_TujQuJe8
5. Next you are going to have to download the update in order to play the game. Go to this link and download the update:
http://halo.bungie.net/images/games/halopc/patch/110/haloce-patch-1.0.10.exe
7.  Install the update then open Halo Custom Edition on your PC/Mac
8. Enjoy the Game!!!!!
9.  If any of you actually play this game regularly I will make my own server and have some Avicus Matches!!

 -IF YOU HAVE ANY QUESTIONS PLEASE COMMENT WITH THEM!!

 Thanks,
