The Tournament Plugin by Tazz December 22, 2015 at 8:12 PM UTC

Hello

Since there is quite a few commands that people are not aware of on Official Servers, I am going to post all of the current commands here, and this post will be updated whenever a new command is added.

If anybody has any suggestions on how to improve the plugin, please post them below. An official will then communicate that suggestion to Kylo or Javi if they think it would be beneficial. 


Perm

Command + Args

Brief Description

Examples

Official

/tm ref <player>

Toggle referee rank for any player.

/tm ref KyloCode

Official

/tm broadcast <message>

Broadcasts something to the whole server

/tm broadcast match starting in 5 minutes.

Ref

/tm register <team tag>

Sets an open team in the match to an Avicus tournament team.

/tm register scm

Ref

/start (time)

Force match to start.

/start 10

Ref

/end

Force end of match.

/end

Ref

/cycle (time)

Cycle to next match.

/cycle 10

Ref

/setnext <map>

Set next map in rotation.

/sn Enoki

Ref

/tm ready <team tag>

Once a team is ready to play, a ref types this for them. The match starts after all teams are ready. This toggles ready status.

/tm ready scm


(type again to unready)

Ref

/rc <message>

Puts a message in referee chat that

/rc Match on Official01 between x and y has started.

All

/team

/t

Toggle team chat.

/t


All

/report -t <player> <reason>

Report players to refs, advisors and officials with the flag

/report -t KyloCode <reason here>
