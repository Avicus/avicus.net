_DC October 26, 2014 at 1:10 AM UTC

As many have mentioned, yes bukkit has not been updated for 1.8 as of yet:
https://storify.com/lukegb/the-tale-of-bukkit-for-minecraft-1-8

So no Bukkit servers have been updated for 1.8 to my knowledge. The reason that you can join these servers on a 1.8 version of minecraft is because of the Handshake packet. The Handshake packet gives the server the version of minecraft the player is on. The server then compares it's version of Minecraft to the client's handshake's version.

To get this "1.8" effect you can simply modify the Handshake code. But if you go on a so called "1.8" bukkit server, you will find that no 1.8 blocks work and no features from 1.8 work on the server.
