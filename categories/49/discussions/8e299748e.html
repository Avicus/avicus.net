Atheyna January 10, 2015 at 1:01 AM UTC

Hey there, and welcome to the forums! Here's a couple things you should know if you don't already :)

Avicus currently has three different donator ranks. Gold, Emerald, and Diamond. Make sure you review the rules, which can be found here http://avicus.net/rules. Currently Avicus has Nebula, Nexus, CTH, Avicade, Walls and Elimination for minigames.

Staff applications are currently open!
To apply you can create a new discussion in the "Staff Applications" section by clicking "New Discussion" at the top right. Please use the format found on this thread http://avicus.net/forums/ef08bacb4 Once you have submitted your application you should wait for a response.
If accepted, you then have to have a TeamSpeak interview, where you are asked various questions by the senior staff to see if you qualify for Moderator. If your interview is up to standards or good, you will be promoted to Jr Moderator and will be put on trial for approximately a month. If you perform excellence in Moderating over the month, you will be promoted to Moderator. If not, you will be demoted.

If you wish to create a map for the server make sure that you have it in a void world and that it is unique! You can upload your completed map to the forums under the "Map Releases" section with a download to the map, pictures of the map and a YAML. You can visit the Avicus YAML docs to learn all about how to create YAML's!
(http://docs.avicus.net/)

If you have further questions, don't hesitate to ask anyone, and I hope you have a wonderful time on the server and the forums :) Hope this helped!
