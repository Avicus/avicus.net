How did you become a Map Developer?
I was a member of the map committee when it first started, and I was asked privately by Gill if I wanted to be in it. I accepted it, and I believe I grew into a mapdev by doing a lot of contributions, eventually me asking if I could have access to the Github repo. I got it, then after a while, I got the user rank, after that Junior Dev (which lets me help manage docs.avicus.net and issue tracker). I'm now a mapdev :3

Why isn't Koodalord a Map Developer?
I don't know. I'm not really the one to ask about this, but I believe he's caught up with his own server right now xD

What's the most important quality to become a map dev?
I believe teamwork. Collaboration is very important instead of just personal opinions and making random changes to the server. We are able to do stuff faster, and better, that way.
