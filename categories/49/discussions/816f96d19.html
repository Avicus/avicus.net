Hey there, and welcome to The Avicus Network! Here's a couple things you should know if you don't already :)

Avicus currently has three different donator ranks. Gold, Emerald, and Diamond. Make sure you review the rules, which can be found here http://avicus.net/rules. Currently Avicus has Nebula, Nexus, CTH, Avicade, Walls and Elimination for minigames. For a full list of current Avicus staff members, visit this link http://avicus.net/staff
