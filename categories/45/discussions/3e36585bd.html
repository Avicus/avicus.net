Advent of Code - for you developers! by Javipepe December 2, 2015 at 5:12 PM UTC

So, an absolutely stunning and really fun project was put up lately by Eric Wastl ( http://was.tl/ ) which has caught my attention and apparently, the one of many other devs around the world. There are two puzzles by which you gain one star (*) each, published each day; you need to solve the first one to see the second, though. 

Though, I may say it may be slightly complicated as days pass, it'll sure be a lot of fun if you know the basics and loops, yes as you heard it, a freaking lot of loops. Basically puzzles can be solved with lots of runnable languages, or sheer logic and straightforward thinking; which one to you is entirely up to you, as long as you get the correct result.

Apparently too, there are different inputs for puzzles (me and you, reader, may have the same heading, but not the same numbers/info), not entirely sure about it though.

If you want to start on it, give it a shot, it's really worth it in my opinion.

http://adventofcode.com/
