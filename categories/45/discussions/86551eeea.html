1. Control Panel -> Uninstall a Program -> search "Hades" and look for any suspicious file names downloaded that you are not aware of.
2. Open Web Browser:
(Chrome): Options -> Settings -> Extensions
(Internet Explorer): Options -> Manage Add-ons, Search engines, etc.
3. Libraries -> This PC -> Program Files/ x86 Program Files -> Search for any folders like "Hades"
