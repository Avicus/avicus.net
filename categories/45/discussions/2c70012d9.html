Project Float - The Reboot by Derumisis May 12, 2015 at 9:05 PM UTC

Hiya there, you may have noticed this post is extremely old, and it was previously a Linux Operating System I was working on. Obviously that was cancelled without notice, as no one ever got a preview in February 2016...

I've rebooted Project Float completely, and I have been working on since October on paper, and finally am bringing it to life... BUT. It's different.

Project Float will be a Raspberry Pi based project, where I will utilize a Raspberry Pi, to run handheld, and for it to be an epic handheld and TV connectable games console which can run for a lengthy amount of time, on the go. I am aware there are like 100s of other projects similar and like this, but I'm working to make mine just the tiniest bit different.

I'm currently building this project off a Raspberry Pi 3B (the latest build as of December 2016), which means it has the best potential.
It can currently run Nintendo 64 games at a smooth fps, and Dreamcast games at a smooth fps, which means it has plenty of potential.

Currently it is in Alpha 2 as of 26/12/2016.
