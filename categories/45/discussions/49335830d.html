Advertiser Rank? by EDJM9 October 1, 2016 at 12:10 PM UTC

Ok, before I start this a bunch of you are probably thinking, another useless rank? Well, how did servers like Mineplex, Hypixel, and other big name servers get popular. They advertised. It might be just me but I feel like the Avicus Server Population is gradually decreasing. I feel a main part of this is that we aren't advertising enough. So this is where the Advertiser Rank comes into play. 

A person with the Advertiser Rank is responsible for "advertising" the avicus network. This can be through social media, Youtube, through forum pages and sidebar ads. 

I think this idea is good and it will benefit Avicus now and in the long run. Avicus's population will increase if we advertise more.

Yay or Nay?

-EDJM9
