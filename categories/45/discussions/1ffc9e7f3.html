FrozenSolstice December 21, 2015 at 10:12 PM UTC

I use PhotoShop, a cracked version. For renders, I use Cinema4D.

As I told you earlier today, you need to download the Minecraft Frost Rig 5.0 so you can have a Minecraft character. Watch the tutorial on Frost's website so you know how to use it. For skins, download them off "Miners need cool shoes". The reason for this is that the Frost Rig only supports 1.8 skin formats, and Miners Need Cool Shoes supports 1.8. Finally, look up some good rendering tutorials on the Internet. Also, a good way to get HD renders is by using a Lightroom. Use Google to find some good ones. I personally use the "Kraytex Lightroom". 

For or the thumbnails themselves, I use a GFX pack which I will link here later. One is mainly MC themed, while ShadowZ's pack focuses on more professional things. As I said earlier, YouTube has good tutorials on thumbnail creations.

Good luck!
