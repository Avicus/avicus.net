Hey everyone, I thought it would be pretty fun to create another forum game!

Basically, I will start by saying a word, and the player below has to say another word which starts with the last 2 letters of the previous word.
For example:
Player 1: orange
Player 2: gesture
Player 3: repeat
And so on...

If you say a word which ends in 2 letters which cannot be the start of a word, that player can choose a new word to start off with (An example of one of these words is rent, since no words start with "nt").

RULES:
  • Must be appropriate
  • Cannot be any names

Have fun! ^o^
