The Place-Name game | Forum Game by Crazy_ February 23, 2017 at 7:02 PM UTC

So the Avicus forums seem like they need a forum game to spice things up. I introduce this super fun, probably repetitive forum game!

Anyways, let's explain how it works. The first person (me), names a place. So for example, Texas. The next person, then has to name a place that starts with the last letter on the place before. So if my place was Texas, the next guy might say Sweden. And so on and so fourth.

Easy enough right?

Now there's just one catch. You can't say a name of a place that has already been said on that forum page. (I'd like to do the entire thread, but it's no fun to have to look through the entire thread to see if you place has been said yet). So again, it can't be a place that has already been said on the same forum page as you're posting to.

The last catch (I lied about there being only one catch), is that you have to try and avoid naming places that end with the letter A (if it ends with an A, that means the next guy has to start with an A, so be careful when naming a place!) If a full page of places are named that begin with the letter A, everyone loses.

So best of luck, the first place is going to be Egypt.
