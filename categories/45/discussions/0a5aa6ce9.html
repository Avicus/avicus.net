Hey guys, so I retired.
Probably not hard to see.
So yeah, I've had a lot of stuff going on in life and I haven't really been enjoying moderating or Avicus in general as much as I used to, so I've decided to call it a day. As some of you may have noticed, I've been inactive af these past few weeks, so I came to this conclusion.
I didn't think I'd ever lose interest in Avicus. Since I found this server with a few friends back in January 2014, I knew that I wanted to stay here. But sadly, it's come to an end. I'll still be here, just not as much as I used to be. But if you ever need some advice, cheering up or just want a cheeky chat, I'm only a Skype message away :).
