vlad, search up the name of the plugin you are using, so they can tp with signs, then type in google "plug-in name" permission nodes.
Then, find the permission node that permits players to use signs to tp.
Using your permissions manager, add this permission for all groups(or the groups you want able to do this)unless your perms manager has inheritance, then you can just give the perms to the lowest group that u want to have this perm.
Hope this helps :)
