I know this was a game already created a very, very long time ago so I thought just best to repost this game as for I'd be "bumping" very old posts.

Game

The game is simple from all the avicus maps (which can be listed HERE) create your own rotation! 

Rules

You must have a rotation name (eg. EPIC ROTATION)
You must state a server on avicus which you want this rotation, basically any server which is or will be in Avicus.
You may not copy another person's name or rotation (or the current rotations)
You must follow what rotations generally consist of.

Rotation Rules

The Walls - 8 Maps
Nebula - 12 Maps
Legacy - 6, 8, 12, 18, 24 Maps.
Other Games - Put 12 in (or if there isn't as many just put as many as you can).
