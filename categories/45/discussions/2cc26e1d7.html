Sevoo February 14, 2015 at 6:02 PM UTC

Get
Ready
For
The
Awesomeness...

PvP class

Basically learn all the different types of PvP servers, tactics and you have to 1v1 the teacher and win for the final exam,

Building

Learning how to ninja bridge, normal bridge, super block spamming (a tactic created by me, or at least I think so, to block spam, but in a way that you are placing multiple blocks on top of each other to jump up walls and make it harder for you to get hit. This would work with many different mouse clicking ways, jitterclicking for instance), breezily bridging and just learning to build beautifully. This would kind of be art class in a way, if you ignore the first ones I said.

Redstone

I don't know much about Redstone, so this would be a hard, yet useful class for me. You would learn howto make machines with pistons, redstone and so on.

Wizardry

This is how to make potions and brew all sorts of potions and that sort of stuff. Enchanting would be something too, so yeah.

Survival/PvE

You would learn how to kill mobs, how much health they have and so on... A good way to practice this is to go on a flat world with creative all the supplies you need (I recommend full diamond and a sword with secondaries), get OptiFine, set the time on night only and put the difficulty on Hard. I do this all the time and it's really fun!

Basics

Learn all the blocks, and the ways to destroy them, their stats  and so on. Also items, what they do and how to craft them.

Controls

Probably the easiest class, but you can customize and stuff. No homework :P.

Resource packs, mods and skins

Learn how to make resource packs, mods and skins. Probably the class with not much homework, but the homework given is in big amounts. This class calls for the most creativity and originality.

If I come up with more ideas I'll post them. :)
