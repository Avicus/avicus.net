If your main goal for learning coding is to develop for Minecraft, don't even bother with HTML, CSS, or any of the web languages. Dive straight into Java. HTML will not help you with Java at all, since they are completely different. HTML is a markup language, while Java is a C-based programming language. Java is one of the easiest languages to learn, so it shouldn't be hard to start off with it.
