[Pagoda UHC] Test Game #2 by Spoookeh May 30, 2017 at 11:05 PM UTC

I'm pleased to announce I will be helping to host UHCs on   KojaCola's new server, Pagoda. This is again only a test game to help eliminate any bugs before becoming a full reddit + private avicus UHC server. 

Discord https://discord.gg/zz678wc

Server information

IP 162.244.165.137 or pagodauhc.mc.gg
Slots 200
Location NA
Version 1.8 x

Match information

Scenarios cTo2, CutClean, Backpacks
Whitelist off May 31st 3pm EST/8pm BST
Whitelist on 5-10 minutes after open
Team Size cTo2
Map Size 1000x1000
PvP Enabled 10 minutes in
Duration 60 minutes until border size shrinks to 100x100

Scenario information

cTo2 Use the command ./team to create a team with you
CutClean All mobs drop cooked meat, and ores drop smelted.
Backpacks Throughout the game, you and your teammates can access an inventory that is shared among your team (./bp)

Hope to see quite a few people there, I can say the server has improved a lot since the last test game.
