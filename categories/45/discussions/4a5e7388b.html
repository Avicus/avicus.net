This is a remade post from before but this has images to help , Im running windows 8.1 but should work with windows 7 . This is just one way there is lots more ! This is a in-depth tutorial on how to find out your ping ! Your ping will vary where you are from your internet and a few other things !Instructions how to is below...

COMMENT YOUR PING BELOW 

Step 1 :
Hold down both the windows key and  R together and a menu like this should pop up . Or you can search in the  windows search menu and search for a program called run.


Step 2 : Now type in cmd into the run window and click ok then this should open up . This is your Command Prompt or known as CMD to learn more about this go here.


Step 3 : Now your CMD has opened up now type in "ping avic.us" .


Step 4 : Then press enter on your keyboard and you should get these results which are the ones you need , my IP address has been blanked out so no-one can see . But the parts circled yellow are the parts your looking for these are your ms (Milliseconds). Now what it means is my average ping there was 152ms , which means it takes 152ms to travel to where Avicus is hosted and then registered . If you have 1400ms ping it means it takes 1.4 seconds .

Thanks for reading if you have any questions comment below if there is any mistakes or errors I have made feel free to comment them down below !
