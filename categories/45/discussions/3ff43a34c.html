Hey, Im here to introduce my Public Scrimmage Server, it has old OCN maps and in the future some Avicus maps (will try to get permissions). This server was created to keep a fraction of the Overcast community. I know Avicus is a 1.8 server, but since there are some OCN players here I need some advertisement to make this work. This is not a network.


                                                                                      s3.hosthorde.com:26330

  Cycle

You can ask for the cycle rank if you are the leader/co-leader of a team and you want to play here. Remember that there are rules, do not abuse it or it will be removed.
Form

  Rules

You can see the rules typing /rules on the server!
Optifine, Togglesprint and Labymod are allowed. If you think someone is cheating you can use /report. This server has a dedicated staff that is almost always online. 
Note: Screenshares will take place if a person recieves too many reports or if he is suspicious.


 Mod Applications coming soon! 
You can join to our Discord to be in touch!


Remember that I do not earn money doing this. 1.9 version.
