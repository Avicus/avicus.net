"No mods online, so there is no point in using /report"
I have heard this alot, and I just want to clean this up abit:
So you may think using /report when no mods are on will not help, as there are no staff on to deal with the reports... This is true obviously, but mods have been able to check reports when not online in game for a while now, so if a staff member does check reports and notices there is a recent report, then they will more than likely come online and sort it, if they have checked. Also even if there is a hacker online, and a mod logs in, if you have not yet reported them, then a mod will either take more time to find out for them self's about it, or the person may log off. As mods do login often, check recent reports, and log out if there are none.
Thanks for reading and report if someone is breaking the rules, even if no mods are online :)
