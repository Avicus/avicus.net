Javipepe August 11, 2015 at 1:08 PM UTC

they both use bootstrap which (I think) is a certain kind of layout/template
Well... it's not exactly that.

To tell you what Bootstrap is, you'd first have to know what HTML and CSS is. HTML makes the whole structure of the webpage, however, it doesn't include colors, widths, margins, icons, dropdowns, etc etc. To see the avicus webpage without css so you can get the idea do the following:

Press CTRL+Shift+J while being on avicus.net and go into sources, go into the folder "assets" and click the first "application" file, and remove all its text, do the same for the other "application" file. Now you are on non-css mode.

CSS lets you personalize and edit every single thing on the html, plus creating your own things, in other words, defining the style of a webpage (hence they are called "CSS -> Cascading Style Sheets"). Bootstrap is a big CSS file that is implemented in the webpage, and it defines some stuff, mainly for responsive (cross-device correct displaying) proposes. So it's not a template, indeed, there are some bootstrap templates that kinda "add" things to bootstrap to make it handy dandy to an editable html file, hopefully that made sense. If you are curious, you can see the main bootstrap file here: https://maxcdn.bootstrapcdn.com/bootstrap/3.3.5/css/bootstrap.min.css

But yeah both OCN and Avicus use bootstrap, but for example, my webpage (http://javipepe.pw) does too, and it ain't similar to any of them
