The Global Game by Flebula September 15, 2014 at 9:09 AM UTC

Basically this is a game I used to play in the car with my siblings when I went on a trip somewhere. 
It works like this: Someone would start off with saying a random country. The next commenter would then have to name another country starting with the letter the last commenters' country was ending with.

Example: 
If I said Turkey as my country, the next commenter would have to reply with a country starting with Y because that is what Turkey ends with. 

Let's see how long this game can go for!

P.S Avoid countries ending in A. When that happens it usually becomes harder to come up with other countries for some reason ._.

Good luck!
