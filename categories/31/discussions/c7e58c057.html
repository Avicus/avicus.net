Translation Update by Posighdun March 28, 2016 at 12:03 PM UTC

So it has been a while since i last addressed everyone about the Translation Team. Over the past month or so the translators have been hard at work translating our rules to various languages. 

Currently, the rules page has been translated to the following languages;
  • Spanish
  • German
  • Swedish
I've also used this time to update the English rules, correcting grammar and such. We currently are also translating the rules page into Dutch.

The translations won't appear on the website until Kylo has pushed it, which should happen sometime this week or next week. 

All the translations are located in the Github Repository, here. If you speak one of the languages mentioned above then i would highly appreciate it if you just read over the rules relating to your language and make sure they are correct. The translators are humans, we make mistakes every now and then. 

If there is a mistake then please sign in to Github and "propose a change" or file the issue under the issue tracker, here

Lastly, i'd like to thank all the current translators for their hard work. Thank you Javipepe, Resepignev, Sayah, Kyler, Justbudah, Pelpelajax & Jiveh.
