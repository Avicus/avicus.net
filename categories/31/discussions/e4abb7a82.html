TheTNTPotato December 3, 2014 at 12:12 PM UTC

Although it does depend on the situation, some of us.. I guess, have a basic mental guideline that we use. It can definitely change, but mine is:

Swearing punishment is ... Verbally warn if minor, kick right away without warning if it's severe.
Using a hacked client punishment is ... Perma-ban. Although we can give long tempbans, I personally do not.
Combat logging punishment is ...  Warn them if I see them. If I do not, I tempban them for an hour.
Being rude punishment is ... Verbal warn -> Kick
Using lots of CAPS punishment is ... Verbal warn -> kick
Team Griefing punishment is ... Kick, tempban if continued. I tempban for 6 hours, but I've seen others do it for up to a day.
Abusing /report punishment is ... I usually give 2 warnings (verbal), then kick.
And spamming chat punishment is ... Verbal warn -> Kick

Not all moderators go by this, we do have a very basic and general guideline thats way different from this. We also have /warn, which I use, paired up with verbal warns. I've never gone by guidelines, I always do things my way. It is usually more lenient than the guidelines I've  been given in the past, but it works. Really, it's completely up to the staff member, as to what they want to do to the person.
