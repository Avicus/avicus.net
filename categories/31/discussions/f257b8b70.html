MasterProX February 8, 2015 at 12:02 AM UTC

I used to have this issue... try editing your skin file so that all the blank spaces you're not using are transparent, not white. Programs like Microsoft Paint (the default on Windows) only make pixels white when you use the eraser tool; you have to use something more advanced — like Photoshop or, for free, Paint.NET — to get them transparent, if that makes any sense.
