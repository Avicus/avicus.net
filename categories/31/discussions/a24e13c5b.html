Well, after researching stuff in the internet about a way using jquery to get the arrays from the api site, I saw that it isn't possible to use jquery xD
There is a problem called "Same Origin Policy" this means that you can't query information of a HTTPS site with a script that is running on a HTTP site (I think that's the problem).
Another problem is that this "same origin policy" is blocking scripts to access the data because a different host (it needs a exact match). So this means you need the api page url like that: http://www.api.avicus.net/v1/status, but that site doesn't exist.
