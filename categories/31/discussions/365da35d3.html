Polar_Express August 23, 2014 at 11:08 PM UTC

`Civilian - A good starter Kit
`Cow - A fun Kit, but the grass does not add much if any to the food saturation level, So once you eat, you pretty much got to keep eating - like a cow, lol.I suggest adding more saturation to the effect of eating grass. ` 
`Miner - Useless - Why would you pick this kit if you could get civilian? This should be upgraded to an Iron Pick. ` 
`Engineer - This is  kit is ok, but also give this class an Iron Shovel - to help make traps and dig so that the engineer can make a base quickly. A wooden bow and arrows or fishing pole would also help this class out and make it more fun. ` 
`Cook - If food is all you need, its fine. I would add a 2nd golden apple to make it a stronger kit and more used.
`Monkey - This is a fun kit - I would add 5 more apples as I cut down every leaf on the walls map ( just the starting quad) and found 1 apple. Also , add leather boots with feather falling 1. I got damaged a few times and when I jumped, and the damage kinda got me off track. It would also be fun to be able to jump from a little higher without getting killed - Nothing stronger than feather falling 1 leather.
`Warrior - Warriors want to go into battle, Add a golden apple to this kit and a chain chest plate. Ready to go.  ` 
`Beast Master - I used the bones and could not tame all the wolves - add 1 more wolf and 15 more bones. That would make the kit useful IMO.  
`Snowman - This is fine as it is.   ` 
`Poseidon -This should be 1 and 1/2 hearts every 2 sec while in water, and should start with a chain Aqua Affinity helmet. The current regen rate makes it where you cannot survive 1 addition hit by an iron sword, making it useless. Adding the AA helmet and increasing the regen rate makes it viable when running away, through water etc to gain some regen. the 1 and 1/2 rate will not give a superior tactical advantage, but it does make it a viable kit.
`Lumber Jack - Good Kit -  add a fishing pole.
`Fire Man - Kit  needs a little help - add an 8 min fire res potion `
`Builder - Add 2 doors , 4 glass, 4 saplings, and start this class with a gold axe.
`Cookie Monster - 5sec buffs kills the use ability of the kit. buffs should be moved from 5 sec to  1:30 sec of random buffs, getting a different buff for 30 sec every 30 sec. Cookies cant be eaten but every 1:30sec.
`Dwarf - This is an ok kit - but should be an Iron Pick
`Healer - This is a pretty useless kit: This kit should modify any healing done. So if you drink a healing potion you get 50% more the effect not just the normal effect. This kit should also start with a 2:15 healing potion.
`Scout - scout should be a passive 10% speed buff ` Tank - The leather should have protection 1 on each piece.
`Assassin - The potion should be a 6:00min potion atleast 
`Leech - This kit should award health based on damage done you get back 1 heart for every 4 hearts delt. OR get a 15 sec regen effect when hearts fall below 7.
`Stomper - this kit should no longer take fall damage
`Wither - This is just a broken kit - having to switch weapons to "maybe" cause it to the wither effect, is not viable in pvp. - Any weapon used should have a 40% change to apply the wither effect.
`Bomber - Need to start with 5 bombs and get 1 bomb back every 45 sec, max of 5.
`Archer - good kit, should start with a bow that has infinity but can break.
`Magician - good kit - no change needed ` Summoner - This is not really a helpful kit - Remove 
`Guardian - Fine as it is, should also start with an Iron sword
`Cowboy - Should start with Iron Horse armor, and 16 wheat 
`Barbarian - Broken - This does not fit - Needs Removed.
