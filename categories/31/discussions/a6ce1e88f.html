Some community updates I think by ViceTechnicolour July 16, 2015 at 8:07 PM UTC

Making this because of transparency reasons and maybe the fact not all of this is known from you all.

 Shop Discounts First, we're starting off with some discount goodies. KyloCode himself said, quote, that he "Made it possible to have sales on the shop... Means Avicus will have a sale soon!". This brings some hope into giving chances for the ones who can't quite make a payment to the server to be able to receive some sweet perks, and for lesser the price. Also, there's speculation that the shop will be revamped, as in the revisions page there was not only possibility for sales on the site, but quote, "Better announcements and shop", "Make discounts red", "Fancy ribbon", and "Support shop products", all within a short timespan. Looking forward to the changes, it'll be great to have a long-awaited sale for the server!

 1.8 Overtaking other Versions In data released, we have the new charts of player versions to see exactly what players play mostly on Avicus. Being the most notorious for "breaking pvp" by the Avicus community and not highly regarded by the Minecraft community in general, the players who use 1.8 have been documented to be within a precise range now of 55,000. For 1.7.2 to 1.7.5, the numbers are actually pretty differing. The number is 10,000, which is roughly 13% of the server's players. For 1.7.9 to 1.7.10, which most Avicus players seem to currently play on, it's at only 19% with 16,000 players documented. This makes 1.8 67% of the playerbase, even if it's discouraged. The statistics are changed from when we first tried to merge to 1.8, so it's interesting to see how the server has grown in that regard.

2 New Gamemodes Two new server gamemodes may make their way to Avicus -- and with that, a contest! Though not released, the next contest is going to be entitled Pluto and encase Capture the Flag, which is a capture the wool hybrid without picking a chest and with a different map layout, similar to that of an automated CTW, and a Free for All gamemode where you fight everyone in your way without alliances or teammates. For both of these, they've hurled their way up to the player favorites and will be further developed by the map developers and the developers themselves.
Pluto and Testing Now with contests mentioned previously, we're wrapping up this contest and opening Pluto very soon. The maps in the contest pool now are going to be tested accordingly and scored by the judges, with best luck to the map authors and contributors.

 docs.avicus.net Updates In regards to website development as well, let's take a look on the docs site: There's been updates recently including documentation for Survival Games, bug fixes, documentation for capture the wool, cleaning up old gamemodes, and a few other edits that weren't included however it's open to see everything that changed for yourself. Keep in mind that it is still on the road to recovery, so anything you request to be done on it can be posted on Avicus issue tracker github to be attended to.

 For recent promotions:

 Rotation Fixes A few rotation updates have occurred on Avicus recently, being primarily bug fixes: the walls maps with spaces have been repaired, the nexus rotation has been fixed from loading other gamemode maps, and the 2 maps have been welcomed into the server, both "Castle in the Rocks", a new TNT map by Pavlo and "Northern Omens", a large and intense map by Zintenka.

 Server Plugin Changes On the agenda, there's been a large scale amount of changes in the development of plugins thanks to our new developer, javipepe, and with the collaboration between him and Kylo development is coasting along. There have been changes on Tab Completion, message enhancements, which now show the rank and team from which you PM someone, some cool color changes in chat, new staff commands and utilities such as report clicking, namehistory, and go enhancements, as well as new command aliases, global punishment messages, a new combat logging detector which sends automatic reports when caught, some trails changes, and particle changes. All of these have been within a very small span of time, and already have had a big impact on how Avicus runs. It's on a streak to be continued, so we'll absolutely see how things run from here on out, and for the better.

 Recent Map Acceptances There have been some maps submitted to the server this week through our new program Trello, and the two maps accepted are Maisosen by ParaSpider and FlyHacks and Sketch Vapor by dilpickel. Both are very wonderful maps and have made their way through before, but there's always a second chance as the server is constantly expanding with different tactics and playstyles. Let's hope for the best!
