Unofficial Lists of Maps Needed by OldManTigerMask July 8, 2016 at 8:07 AM UTC

Introduction

I thought I should make it clear, I AM NOT a staff member or a map developer so please don't count this as an official post, this is just me spreading some suggestions for maps throughout the community as for maps are needed all the time but some games specifically require more (in my opinion) as for there aren't as many maps in some of the rotations. 
This thread you may find helpful if you read the latest Map Update 

Maps Needed for each game (My Opinion):
Rotations Generally have 12 Maps, (just making it clear).

Destroy

Destroy the Monument (24/28)
Rotation 1:
Currently has 11 maps and it would be nice to have 3 more maps to complete 14 maps in a rotation
Rotation 2:
Currently has 13 maps and it would be nice to have 1 more map to complete it's rotation.

Team Death Match (11/12)
Rotation 1: 
Currently has 11 maps but I think I could have another map or two. Although I wouldn't put it on my URGENTLY IN NEED list.

Capture (12/14)
Rotation 1:
Currently has 12 maps, yet doesn't have a lot of CTF maps, so CTF maps are needed
 
Other Games:

Nexus (18/24)
Rotation 1:
Currently has 18 maps and a new map isn't mandatory! However I'm sure the Map Devs would be more than happy to have extra maps for the Nexus game mode considering they love the 6 times tables :P!

The Walls (19/24)
Walls rotations are 8.
Rotation 1:
Currently has 8 maps and it's not mandatory to add to this rotation in my opinion and besides rotation 3, requires a lot more maps.
Rotation 2:
Currently has 8 maps and it's not mandatory to add to this rotation in my opinion because rotation 3, requires a whole lot more maps.
Rotation 3:
Currently has 3 maps and I believe its very likely that server 3 won't be played because of the lack of maps in the rotation so I suggest a few more maps would be handy for this rotation.

Control the Hill (11/12)
Rotation 1:
Thankfully this game has quiet a few more maps however I think it would be nice to receive another map to make it 12/12, a complete rotation. Like most rotations this isn't mandatory but I think it'd be nice to have one more.

Survival Games (15/18)
Rotation 1:
Has a decent amount of maps however since it is Survival Games, I'm certain the Map Developers would appreciate a lot more maps so they can have 2 more rotations for servers SG02 & SG03.

The Lobby
Rotation 1:
Doesn't need another map :P but you can try I guess XD. 


Conclusion:

Check out the Map Making FAQ for more information on how to make maps

Latest Accepted Map: JungleBlitz
Latest Denied Map: WavesDTM 

Thanks for reading this all you kind folks and I hope this is useful to the talented Map Makers and that you will pick something new from this list. Anyways I will continue to update this as the map rotations update and new maps come in! 

Hope you found this useful,
OldManTigerMask

ps. If I've put this in the wrong category please move it :P
