Rule Change - Public Hackusations. by Joao_Victor July 4, 2014 at 7:07 PM UTC

Hello Avicus.

In case you haven't heard yet, accusing a player of hacking through chat is now a punishable offense.

You can now report players for doing this, however, one hackusation is not enough for a temporary ban, so if you report someone, make sure it is either excessively spammed, caps were used, or language was used.

(http://avicus.net/rules)

Thanks
-Nolan
