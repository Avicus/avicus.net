I'll go from left to right explaining each thing. At the very left there's just the Docky configuration for the actual dock, notice I don't use mac but linux, I just like that docky thing. Then there is the file explorer, nothing much to it. Then minecraft.jar, that's how I launch it (there are several ways of doing it on linux though). Next is teamspeak, nothing to explain here :p. Now there's mumble (somewhat hing similar to teamspeak, more info at http://www.mumble.com/ ). Next is IntelliJ Idea Community Edition, that is the software I use to code, it's a great Java (along with many other languages) IDE, my preferred. Then there's Atom which is the software I use for HTML, HAML, CSS, and javascript (aka front-end/website stuff). Next is shutter that is kind of a gyazo thingy for linux, takes screenshots and uploads it to whatever place you want or if preferred, by FTP. Then terminal, just that black scary hacky thing, I just change the color of it, and git commit :p. Then there's my calendar, something I personally set up with Google Calendar API and many other things, nothing you can download sadly :((. Now there's Filezilla that is the p2p FTP client I use, just for uploading  files to my personal testing server and some website stuff, nothing much to it either. Then there's the google chrome launcher, that is my main browser. Then there's SSR (simple screen recorder), which is the recorder I use; it uses OpenGL injection stuff, I got helped by an ocn staff member on how to set it up properly, and I fell in love with it!
