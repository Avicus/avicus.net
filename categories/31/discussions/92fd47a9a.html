Introducing LeagueRec! by Matic0B February 26, 2017 at 3:02 PM UTC

Hello everyone,

Recently the ResourcePile team has been working to add more content and create more for the community. So we are introducing LeagueRec.

LeagueRec is a collection of information about competitive events that have been hosted by various networks. We aim to archive information about these events because some networks remove access to the information after a tournament has finished.

With the start of the project, we have added information about all events ran by the Avicus Network which you can view by clicking on the link below.

Avicus Network - League Rec

If there are any issues you can create an issue or pull-request on our Github repository (please read our documentation). 

Hopefully we can add more content from other networks in the future. 

This is not associated with any official Avicus services.

What's resourcepile
Its a community project with which we (ResourcePile team) wish to improve the experience of users that play on Minecraft networks and help them by providing useful resources. It includes StatTrack (tracking stats like bow shot records), MapStack (collection of publicly available maps), toolrack (collection of helpful tools which is at the moment WIP) and leaguerec.
