[Early Stages] Free for All now a go by ViceTechnicolour July 5, 2015 at 12:07 PM UTC




I noticed that there was a free-for-all module that was hanging around somewhere in another file for another gamemode, and to represent some more new changes to the server I'd like to incorporate FFA into Avicus. As I really don't want this project to die off, I want to have everyone pitch in for maps, and ideas. Maps will be strictly considered as FFA is a tricky gamemode to build for. I will release specifics of these right now, so you can know what you're doing:

Maps
- Must be specifically designed for 12 - 16 players
- Must be oriented around Nebula TDM
- Must have cover and levels and gameplay in mind
- Must have escapes and dangers to a player
- Fall damage is disabled
- Team colors are either white or yellow
- Must use multiple spawnpoint feature
- Every map uses same loadout

Ideas
- Be constructive about it
- Try to say positives before negatives
- Don't yell "copy copy copy". We're going to spin to originality

We have one map now, called "Block Fortress" by timaeusTestified.

The only bug currently is in need of fixing is match ending (it doesn't force end, it confuses the server) and to find a better way to distribute points. Otherwise, this gamemode is good! As long as it's under mapdev supervision, it can be played and enjoyed.
