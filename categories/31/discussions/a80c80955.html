DTM Map Submissions Update by ProfessorUtonium June 16, 2016 at 3:06 AM UTC

Hey Guys!

The Map Development team will no longer be accepting DTM maps until 2 weeks have passed. Please be aware of this. You may still submit DTM maps after that time, but do not expect a reply from a map developer. We are doing this because we have a surplus of DTM maps and we would like to start having more maps other than DTM.

If you are a frequent DTM map builder, please try to build maps for some of our other gamemodes. These include CTW, TDM, KOTH, CTF, Lanes, or FFA. Those maps will be looked at. Any DTM map submitted before the due date will still be looked at and will have the ability to be accepted or not. So if you are currently working on a DTM map, start working and publish it asap!

If you have any questions, respond below.

Regards,
The Map Development Team.
