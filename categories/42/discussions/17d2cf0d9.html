Adventure map Contest by ClubberNugget July 16, 2015 at 7:07 PM UTC

Hello Everyone

Are you ready for a different challenge?

Do you want to try and build something you haven’t before?

Do you like credits?

Well then I have a Contest for you!

Hello all, Random here, and I have come to announce that I will be holding an adventure map building contest. 

What is the prize?:

As of now, 27,000 credits, but it will probably be 30,000 by the end because I’ll have the credits to provide that.

What is an Adventure map?

An adventure map is a map built in Vanilla minecraft, usually using command blocks, to create a mini map that another player can download and try to beat, like a dungeon for example, you can use parkour, custom mobs, puzzles, for each level.  You usually are on adventure mode when you play these maps.  If you still do not quite understand, here are some gameplays of adventure maps.

How do I submit my map?

Just post it below! Make it look like it’s your own thread if you’d like, just make sure to provide a download link and a brief description of the game you created!

How is a winner determined?

I will play each game; I will record each one and post it on my Youtube channel.

Once the deadline has been met, I will score each map submitted like so:

1-5 points for aesthetics, who doesn’t like a nice looking map?

1-5 points for storyline, an intriguing story can make the map all the better

1-10 points for gameplay, this one is special because this is more important than aesthetics and storyline. How smoothly the game progresses, how well the levels are made, how fun each level is, all can decide if you like the map or not, storyline and aesthetics enhance the experience but the gameplay makes or breaks a map.

Once I decide a winner, I will update this thread and post a video declaring the winner and everybody’s scores.

What version can I use?

I will be using 1.8.7 so I suggest that you use it too to build your map.

When is the starting date and the ending date?

It has already begun! The last day you can submit a map is September 1st.  This is a long time away but Adventure maps take a while to make.

Is this an official Avicus contest?

No, this is a contest being run by me, myself and I, (aka: only me) and has no affiliation with the map development team here on Avicus.  But the prize is Avicus related, if I didn’t use credits, then I would have to nothing to use as a prize, plus, most of these credits were won in a giveaway so I felt I don’t deserve to use them to buy a badge, plus this can be interesting if you participate.

 

Any other questions may be asked below; best of luck to you all who plan to participate!
