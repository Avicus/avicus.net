Basic Commands:


"<Requeried>, (Optional)"
- /sn,setnext <Map> - set next a map
- /cycle <Seconds> - cycle to the map that has been set next
- /recycle <Seconds> - recycle to the map that was just played
- /start <Seconds> - start the match
- /end - end the match
- /cancel - cancel a countdown
- /maps - view all maps on the server
- /report <Player> <Reason>
- /ping (Player)
- /freeze <Player>
